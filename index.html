Preguntas frecuentes

Todo lo que necesitas saber antes de empezar.

¿Mis datos de pago y los de mis clientes están seguros?

SuscripFy nunca almacena datos de tarjeta (número, CVV, fecha). La tokenización ocurre directamente entre el navegador del cliente y Wompi mediante su SDK. Solo guardamos tokens opacos que son inútiles sin las credenciales del merchant, las cuales están encriptadas en nuestra base de datos.

¿SuscripFy afecta el rendimiento de mi tienda Shopify?

No. SuscripFy opera como un servicio externo que se comunica vía API. El widget se carga de forma asíncrona y pesa menos de 15KB. Si SuscripFy estuviera caído, tu tienda sigue funcionando normalmente con su botón de compra nativo.

¿Qué pasa si un cobro automático falla?

SuscripFy tiene un sistema de reintentos inteligentes. Si Wompi rechaza un cobro, el sistema reintenta automáticamente en los días siguientes con notificaciones al cliente por correo para que actualice su tarjeta. Solo después de agotar los reintentos se marca la suscripción como fallida.

¿Cómo garantizan que no haya doble cobro?

Cada cobro usa una clave de idempotencia única en Wompi. Si por alguna razón el sistema intenta cobrar dos veces el mismo ciclo, Wompi rechaza el duplicado automáticamente. Además, el estado del ciclo se actualiza en base de datos antes de intentar el cobro.

¿Funciona con cualquier tema de Shopify?

Sí. El widget de SuscripFy es compatible con todos los temas de Shopify, incluidos los de la Theme Store y los personalizados. Se inyecta vía Script Tag y se adapta automáticamente al estilo de tu tienda. No requiere modificar el código del tema.
